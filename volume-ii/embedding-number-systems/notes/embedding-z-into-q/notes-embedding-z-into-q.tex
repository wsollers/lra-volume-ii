\section{Embedding \texorpdfstring{\(\mathbb Z\)}{Z} into \texorpdfstring{\(\mathbb Q\)}{Q}}
\label{sec:embedding-z-into-q}

\begin{topicbox}{From \texorpdfstring{\(\mathbb Z\)}{Z} to \texorpdfstring{\(\mathbb Q\)}{Q}}
\begin{exposition}
The rationals are constructed so that division by nonzero quantities can be
represented internally. The integer \(a\) is embedded as the rational class
represented by \(a/1\). This embedding preserves addition, multiplication,
negation, and order. It lets the integer line be read as a regularly spaced
subsystem of the rational line, while the rational system fills in fractional
points between integer points.
\end{exposition}
\end{topicbox}
