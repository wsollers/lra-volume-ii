\section{Intervals on a Number Line}
\label{sec:intervals-on-a-number-line}

\begin{topicbox}{Intervals on a Number Line}
\begin{exposition}
Interval notation depends on the ambient ordered system. The expression
\([a,b]\) means the elements of the current system that lie between \(a\) and
\(b\), including the endpoints. In \(\mathbb Z\), this interval contains only
integer points. In \(\mathbb Q\), it contains rational points. In
\(\mathbb R\), it contains all real points between the endpoints.
\end{exposition}

\begin{exposition}
This is another place where embeddings matter. When an integer endpoint is
used in a rational or real interval, the endpoint is being read through the
appropriate embedding. The notation is the same because the embedding
preserves the order and arithmetic attached to that endpoint.
\end{exposition}
\end{topicbox}
