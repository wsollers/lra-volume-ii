# Legacy Orphan Proof Archive

Files here were moved out of canonical `proofs/**/*.tex` discovery during the
Volume II proof-stub invariant rollout.

Content was preserved. These files were archived because they either lacked a
conservative theorem target, duplicated an active canonical proof stub, or used
legacy naming that could not be safely attached to a current theorem id.

Canonical active proof files remain under `volume-ii/**/proofs/` and associate
to theorem statements with `\LRAProofFor{...}`.
